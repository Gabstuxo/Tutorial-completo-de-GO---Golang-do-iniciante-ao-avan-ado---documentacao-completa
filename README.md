# Golang-guia-completo-ptbr

Guia completo de Go (Golang) em português, do zero ao avançado. Cobre fundamentos, tipos, structs, interfaces, concorrência com goroutines, APIs REST, JSON, HTTP, CLI e boas práticas profissionais. Exemplos comentados e prontos para rodar. Ideal para iniciantes e desenvolvedores migrando de outras linguagens.
